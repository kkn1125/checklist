# CheckList

## 개요

진행하는 작업에 대한 체크리스트를 커스터마이징하고 진행률을 시각화하는 애플리케이션

## 목적

특정한 업무나 진행도를 나타낼 필요가 있는 작업을 수치화시켜 보기 쉽게 하기 위함

## 기능

1. 체크리스트
   1. 등록
   2. 삭제
   3. 그룹 지정
   4. 완료
   5. 완료 취소

## 개선사항

1. 체크리스트 수정
2. 그룹 펼치기/접기
3. 정렬
4. 그룹, 내용 별도 작성

## 사용법

1. 데모 사이트 열기
2. `내용` || `@{그룹명} 내용` 형태로 작성
3. `@{}`문법은 내용의 앞, 뒤, 중간 어디에서든 가용
